Tentang

Hai, nama saya Indra Sah Noeldy Girsang. Biasa dipanggil Indra. Saya adalah seorang developer berumur 14 tahun asal Sumatera Utara (lebih tepatnya di Kabupaten Dairi). Saya lahir pada 7 mei 2009 di Sidikalang (Ibukota Dairi). Sekarang saya tinggal di Desa Tanjung Beringin, Kecamatan Sumbul bersama ibu dan kedua saudara saya (1 saudara saya lainnya sedang berada di Palembang). Saya bersekolah di SMPN 2 Sumbul di Kelas 9.

Saya memiliki keterbatasan, yaitu tidak dapat berjalan. Ini terjadi sejak saya SD kelas 4. Tapi, walaupun begitu, saya tidak putus asa dan tetap semangat untuk meraih cita cita saya yaitu seorang Developer dan Content Creator yang sukses.
