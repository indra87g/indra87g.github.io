Shrinkme Bypass

Ingin mengakses URL tertentu, namun dihalangi oleh Shrinkme?

Tenang, gunakan tools ini untuk melakukan Bypass

Bagaimana cara kerja nya?

Simple. Tools ini memanfaatkan celah yang ada pada link shrinkme, yaitu parameter url

Contoh: https://shrinkme.io/st?api={random}&url={link asli sebelum diubah menjadi link shrinkme}

Dari sini bisa disimpulkan bahwa link asli tidak benar benar diubah menjadi link shrinkme, tapi dipindahin ke parameter url. Hal ini tentunya membuat orang awam mudah tertipu dan mengikuti link iklan menyebalkan dari shrinkme.

CATATAN: Tools ini masih bisa berfungsi selama developer Shrinkme masih belum menyadari celah ini dan memperbaikinya!
